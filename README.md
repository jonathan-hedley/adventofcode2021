This repo contains my solutions to the [Advent of Code 2021](https://adventofcode.com/2021) using primarily [python](https://www.python.org).

## Setup

I'm using python ~3.10~ 2.7.16.

Python requirements (if there are any) can be installed with `pip install -r requirements.txt`.

## Running Solutions/Tests

Any tests I write will likely just be asserts in each test solution file.

### python

- `cd dayXX`
- `python solution.py`
- `python test_solution.py`

or

- `python dayXX/solution.py`
- `python dayXX/test_solution.py`
